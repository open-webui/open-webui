# سياسة البيانات والخصوصية — Agent Workspace

## مبادئ عامة

1. **أقل قدر من الامتيازات**: تفعيل ما يلزم فقط من الأدوات والصلاحيات.
2. **تقليل البيانات**: عدم حفظ بيانات غير لازمة للمهمة.
3. **عدم تضمين الأسرار في Git**: أي ملفات أسرار تحفظ محليًا فقط.

## التخزين

- التخزين المعتمد: `local`.
- الملفات ترفع إلى مسار uploads المحلي داخل بيئة التشغيل.
- يمكن التحول لاحقًا إلى S3/GCS/Azure عند الحاجة التشغيلية.

## الاعتمادات

- تُحفظ الاعتمادات الحساسة خارج الأكواد المصدرية قدر الإمكان.
- في هذا المشروع: `.secrets/` مستبعد من Git.

## السجلات والمراقبة

- يتم استخدام تتبع تشغيلي لأغراض الموثوقية والتحسين.
- يجب عدم كتابة مفاتيح API أو كلمات مرور في السجلات.

## الاحتفاظ والحذف

- حذف الملفات المؤقتة عند انتهاء الحاجة.
- مراجعة دورية للملفات/المخرجات المخزنة لتقليل الأثر.
