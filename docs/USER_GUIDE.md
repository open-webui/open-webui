# Guide de l'Utilisateur — StoryWeaver

Bienvenue dans StoryWeaver, votre cockpit d'écriture augmenté par l'IA. Ce guide vous explique comment tirer le meilleur parti des outils de la suite.

## 1. Démarrer un Projet

Tout commence sur la page **Romans** (`/storyweaver`).
1. Cliquez sur **Nouveau Roman**.
2. Donnez-lui un titre et une description rapide.
3. Une fois créé, cliquez sur **Sélectionner** (✍️). 
   > **Important** : Un roman doit être "sélectionné" pour que l'IA puisse accéder à sa Base de Connaissances dans le Chat.

## 2. Construire sa Base de Connaissances (KB)

L'expertise de l'IA dépend de la qualité de vos notes. Dans l'onglet **Base de Connaissances**, remplissez :
- **Personnages** : Traits, buts, secrets.
- **Lieux** : Descriptions sensorielles, ambiance.
- **Univers** : Lois de la magie, géopolitique, culture.
- **Éphéméride** : Pour ne pas perdre le fil de la chronologie.

## 3. L'Éditeur de Manuscrit

C'est ici que la magie opère. L'éditeur propose 4 modes :
- **Standard** : Liste des chapitres à gauche, texte à droite.
- **Focus** : Masque tout le superflu. Idéal pour "entrer dans la zone".
- **Recherche** : Ouvre un panneau de recherche globale à droite. Un doute sur la couleur des yeux d'un personnage ? Cherchez-le instantanément sans quitter votre phrase.
- **Outline** : Une vue d'ensemble de vos chapitres pour vérifier le rythme de l'histoire.

## 4. Les Assistants Créatifs

Utilisez les **commandes slash** dans le chat (ou via la barre flottante de l'éditeur) :
- `/brainstorm` : "Donne-moi 3 idées de complications pour cette scène."
- `/coherence` : "Est-ce que l'objet que j'ai décrit au Chapitre 2 est cohérent avec la loi de la magie n°3 ?"
- `/dialogue` : "Aide-moi à rendre ce dialogue plus tendu entre Elara et Kael."
- `/outline` : "Propose-moi une structure pour le prochain chapitre."

## 5. Exportation

Une fois votre premier jet terminé, utilisez les boutons d'exportation (haut-droite) :
- **PDF (Format Livre)** : Pour une lecture confortable sur liseuse ou une impression de travail.
- **Markdown** : Pour conserver vos données dans un format ouvert et pérenne.

---

### 💡 Le conseil d'écrivain
> L'IA est votre partenaire de sparring, pas votre nègre littéraire. Utilisez `/brainstorm` pour débloquer le syndrome de la page blanche, mais gardez toujours le dernier mot sur le style et l'émotion.
